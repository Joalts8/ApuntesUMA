Historia de HTML

Entre 1989 y 1991, HTML Fue creado por Tim Berners-Lee, mientras trabajaba en el CERN, con el objetivo de compartir documentos científicos a través de la web. En 1993 aparece la primera versión pública de HTML

EN 1995, Se publica HTML L 2.0, la primera versión oficial estandarizada. Surgen HTML 3.2 y HTML 4.01, con más soporte para imágenes, tablas, formularios y estilos.

Entre 2000 y 2014,XHTML intenta hacer HTML más estricto, pero no se populariza completamente.

En 2014, Se lanza HTML5, que incorpora audio, video, gráficos, animaciones y mejores funciones para aplicaciones web modernas.

  • Lenguaje de marcado: Usa etiquetas para estructurar el contenido (no es un lenguaje de programación).
  • Estructura páginas web: Organiza textos, imágenes, enlaces, listas, tablas y formularios.
  • Multimedia: Permite insertar imágenes, audio y video.
  • Hipervínculos: Conecta páginas entre sí mediante enlaces.
  • Estándar web: Es un lenguaje universal, definido por el W3C.
  • Compatible con todos los navegadores: Funciona en Chrome, Firefox, Edge, Safari, etc.
  • Accesible: Facilita la creación de sitios web accesibles para personas con discapacidad.
